청소비서 팀장 앱 사용설명서 01 / 13 — 신규 배정 처리 순서
새 배정이 왔을 때 팀장이 처음부터 끝까지 처리해야 할 순서입니다. 이 흐름대로 진행하면 빠짐없이 완료됩니다.
1
📬
배정 알림 확인
내비게이션 배정 배지 숫자 확인. 대시보드 오늘 일정에 새 카드가 나타납니다. 빨간 태그(신규배정)가 붙어 있으면 확인 전 건입니다.
2
📋
접수 상세 열기
카드를 눌러 상세 팝업을 엽니다. 위에서 아래로 전체 내용을 천천히 읽습니다. 제목·특이사항 반드시 확인.
3
🏠
고객·현장 확인
주소, 방화 구조(3방 2화), 건물 유형(아파트/빌라), 평수를 확인합니다. 특이사항 메모를 읽어 작업 준비를 합니다.
4
📞
해피콜 전화
고객 연락처를 눌러 전화합니다. 작업 일정을 안내한 뒤 팝업 하단 「해피콜 완료」 버튼을 눌러 처리합니다. 마감: 작업일 전날.
5
👥
팀원 확인
투입 팀원 목록과 연락처를 확인합니다. 필요한 경우 파란 번호를 눌러 팀원에게 직접 연락합니다.
6
⏰
미팅 시간 설정
팝업 「미팅시간」 칸에 집합 시간을 입력하고 저장을 누릅니다. 팀원 전원의 크루 앱에 자동으로 시간이 공유됩니다.
7
📷
청소 전 촬영
현장 도착 후, 청소 시작 전 현장 상태를 사진으로 찍습니다. 팝업 하단 파란 「청소 전 촬영」 버튼을 누릅니다.
8
🧹
청소 진행
팀원들과 청소를 진행합니다. 현장에서 추가 비용이 생기면 결제금액 섹션 → 추가결재 「펼치기」 를 눌러 항목을 등록합니다.
9
📸
청소 후 촬영
청소가 끝난 후 깨끗해진 현장 사진을 찍습니다. 초록 「청소 후 촬영」 버튼을 누릅니다.
10
✅
완료 처리 + 결제
검정 「현장 검수·청소완료」 버튼을 누릅니다. 카드 결제가 필요하면 녹색 테두리 「카드결제」 버튼도 처리합니다.
💡 빠른 요약: 배정 확인 → 상세 읽기 → 고객 해피콜 → 팀원·미팅시간 설정 → 현장 출발 → 전 촬영 → 청소 (추가비용 발생 시 추가결재 등록) → 후 촬영 → 완료 버튼 → 카드결제
이 10단계를 지키면 관리자 화면도 실시간으로 업데이트되어 별도 보고 없이도 진행 상황이 공유됩니다.

1
🚨 빨간 박스 — 해피콜 미완료 알림

화면 상단에 빨간 박스가 보이면 고객에게 전화를 아직 못 드린 건이 있다는 뜻입니다. 마감 초과는 이미 작업일을 지났는데 전화를 못 한 건, 마감 전 미완은 아직 시간이 남아 있는 건입니다. 숫자를 직접 누르면 해당 건 목록으로 이동합니다.

2
8칸 숫자 카드 — 내 배정 상태별 현황
  • 전체 — 나에게 배정된 모든 건의 합계
  • 오늘 — 오늘 작업 예정인 건
  • 분배 — 팀원 배치까지 완료된 상태
  • 진행 — 현재 현장 청소 중인 상태
  • 완료 — 청소 마치고 완료 처리된 건
  • 접수 — 신규 접수, 팀원 배치 전
  • 보류 — 잠시 미루거나 확인 중인 건
  • 취소 — 고객 요청 등으로 취소된 건
3
오늘 일정 카드 — 파란 전화 버튼으로 바로 연결

카드 오른쪽 위 파란 전화 버튼을 누르면 고객에게 바로 전화가 걸립니다. 카드 안 어디든 눌러 상세 팝업을 열 수 있습니다.

4
🔔 오른쪽 위 알림 버튼

빨간 숫자 배지가 있으면 읽지 않은 변경 이력이나 공지가 있다는 뜻입니다. 눌러서 확인하세요.

💡 해피콜 처리 흐름
빨간 박스 숫자 누르기 → 해당 카드 열기 → 고객에게 전화 → 다시 카드 열어 「해피콜 완료」 버튼 누르기. 이 순서로 하면 미완 숫자가 즉시 줄어듭니다.

상태 드롭다운 — 원하는 상태만 보기

전체·대기·접수·분배·진행·완료·보류·취소·C/S 등 중 하나를 선택합니다.

4
검색창 — 이름·번호·주소·접수번호로 검색

5
카드 아무 곳이나 누르면 — 상세 팝업 열림

카드 전체가 클릭 가능합니다. 고객 정보·팀원 연락처·결제 금액·현장 버튼은 모두 상세 팝업 안에 있습니다.

1
맨 위 헤더 — 고객명 · 접수번호 · 상태 배지

팝업 맨 위에 고객명과 SK로 시작하는 접수번호가 있습니다. 관리자에게 전화할 때 이 번호를 알려주면 빠르게 건을 찾을 수 있습니다.

2
담당 팀장 섹션 — 📞 아이콘 = 내 전화번호 확인

메인 담당으로 표시된 이름이 나 자신입니다. 옆 📞 아이콘을 누르면 내 등록 연락처를 확인할 수 있습니다.

3
접수 정보 섹션 — 담당 마케터 「전화걸기」

마케터 이름 옆 전화걸기를 누르면 담당 직원에게 바로 전화가 걸립니다.

4
제목 섹션 — 관리자가 입력한 메모

"오늘·신규배정(미확인)", "4인투입", "사이청소" 등 작업 특성. 반드시 읽고 현장에 가세요.

5
고객·현장 섹션 — 연락처 누르면 바로 전화

결제 금액 섹션에서 추가결재를 등록하면 관리자 화면과 급여 정산에 자동으로 반영됩니다.
1
결제 금액 섹션 — 기본 구조 이해
  • 총 결제금액 — 계약 원금 (변경 안 됨)
  • 예약금 — 이미 선결제된 금액
  • 잔금 — 현장에서 받아야 할 금액 = 기본 잔금 + 회사입금 추가결재
2
추가결재 「펼치기」 버튼 — 현장 추가비용 입력 폼

결제 금액 섹션 하단의 「추가결재 (별도 정산)」 항목을 누르면 입력 폼이 펼쳐집니다. 여기서 추가 항목 이름과 금액을 입력합니다.

3
항목명 + 금액 입력 → 수금 방식 선택 (필수) → 저장
  • 항목명 예시: "발코니 특수청소", "에어컨 청소 추가", "심한 곰팡이 처리"
  • 금액은 숫자로 입력. 빠른 버튼 (+1천 / +5천 / +1만 / +10만)을 눌러 더하면 편합니다
  • 수금 방식을 반드시 선택해야 저장됩니다 (선택 안 하면 저장 버튼이 안 눌림)
4
수금 방식 2가지 — 반드시 정확히 선택
  • 현장결재 (본인이 받음) — 팀장이 현장에서 직접 고객에게 현금·계좌이체로 받는 경우. 잔금 합계에는 포함되지 않음 (이미 받았으므로). 급여 정산 시에는 포함됨.
  • 회사입금 — 고객이 나중에 회사 계좌로 입금하는 경우. 잔금 합계에 더해짐. 팀장이 직접 받지 않음.
5
저장하면 즉시 반영 — 수정·삭제도 가능

저장 즉시 접수 변경 이력에 자동 기록되고 관리자 화면에도 실시간으로 반영됩니다. 잘못 입력했으면 항목 옆 수정 또는 삭제 버튼을 누릅니다.

고객 서명서명 박스에 고객이 직접 손으로 서명. 「서명 저장」 버튼 누르면 저장.
1
기본사항 4개 질문 — 팀장·고객 각각 답변

① 전날 연락을 받으셨습니까? ② 추가 비용 안내를 받으셨습니까? ③ 약속된 시간에 시작되었습니까? ④ 쓰레기를 봉투에 담아 정리하였습니까? — 팀장 먼저, 고객 순으로 예/아니오를 선택합니다.

2
고객 이메일 입력 (필수)

완료본 PDF를 받을 이메일 주소를 정확히 입력합니다. 고객과 함께 확인하며 입력하세요. 완료 후에는 수정이 어렵습니다.

3
동의 항목 체크 — 「전체 동의」 버튼 활용

동의 항목 위 「전체 동의」 버튼을 누르면 필수 항목 전체가 한 번에 체크됩니다. 고객에게 각 항목 제목을 읽어 주고 「전체 동의」를 누르게 하세요.

4
서명 박스에 고객이 직접 서명 → 「서명 저장」

서명 박스를 고객에게 건네 손가락으로 직접 서명하게 합니다. 서명 후 「서명 저장」 버튼을 눌러 저장합니다.

5
「청소완료 (고객 확인·서명)」 버튼 — 최종 완료

확인 창에서 「확인」을 누르면 현장이 「완료」 상태로 변경됩니다. 완료본 PDF가 고객 이메일로 자동 발송됩니다. 완료 후에는 수정 불가합니다.

6
완료 후 — 「고객 열람 링크 복사」

완료 처리 후 「고객 열람 링크 복사」 버튼이 나타납니다. 이 링크를 고객 문자로 보내면 고객이 사진·서명·체크리스트를 스스로 열람할 수 있습니다.

⚠ 완료 후에는 수정이 불가합니다
완료 처리 후 내용을 바꾸려면 관리자가 직접 「무효 처리」 후 재작성해야 합니다. 이메일·서명·동의 항목을 반드시 고객과 함께 확인한 뒤 완료 버튼을 누르세요.

📅 11 · 스케줄
달력에서 날짜를 누르면
그날 일정이 바로 펼쳐집니다
스케줄 메뉴는 달력 뷰와 날짜별 상세 뷰, 두 가지를 함께 제공합니다.
1
연도·월 드롭다운으로 다른 달 이동

상단 드롭다운에서 연도와 월을 바꾸면 해당 달의 달력이 나타납니다. 다음 달 일정을 미리 확인할 때 유용합니다.

2
날짜 아래 청록 숫자 — 배정 건수

날짜 아래에 건수가 표시됩니다. 예: 25목 아래 「5건」이면 그날 5곳 현장 작업이 예정되어 있다는 뜻입니다.

3
날짜 숫자 자체가 청록색 — 손없는날

날짜 숫자가 청록색이면 손없는날입니다 (음력 끝자리 9·0). 이사·입주가 몰려 청소 수요가 높습니다.

4
날짜 누르면 — 달력 바로 아래에 상세 카드 목록 펼침

날짜를 누르면 파란 테두리로 선택 표시가 되고, 달력 아래에 바로 그날의 배정 건 카드 목록이 펼쳐집니다. 고객명·연락처·주소·팀원·시간이 한 카드에 모두 표시됩니다.

5
상세 카드에서 할 수 있는 것
  • 카드 오른쪽 전화 버튼 → 고객에게 바로 전화
  • 카드 아무 곳이나 누르기 → 상세 팝업 열기 (팀원·결제·촬영 등)
  • 상태 배지로 해피콜 완료 여부·작업 진행 상태 확인
💡 스케줄 vs 배정목록 차이
스케줄 → 달력 뷰, 어떤 날이 바쁜지 한눈에 파악
배정목록 → 목록 뷰, 검색·필터로 특정 건 빠르게 찾기

이 날을 휴무일로 지정할까요?
🚫 12 · 휴무일 설정
쉬는 날을 직접
달력에서 설정합니다
휴무일을 지정하면 관리자 캘린더에 바로 반영되어 그날 배정을 피할 수 있습니다.
1
연도·월 선택 — 원하는 달로 이동

상단 드롭다운에서 연도와 월을 선택합니다. 다음 달이나 그다음 달도 미리 설정할 수 있습니다.

2
쉬고 싶은 날짜 누르기 → 확인 창 → 「휴무 지정」

달력에서 날짜를 누르면 확인 창이 뜹니다. 「휴무 지정」을 누르면 그 날짜가 빨간 원으로 표시됩니다.

3
이미 빨간 날짜를 다시 누르면 — 휴무 취소

이미 설정된 휴무일을 다시 누르면 「취소 확인」 창이 나타납니다. 확인을 누르면 일반 날짜로 돌아옵니다.

4
저장 버튼 없이 즉시 반영

「휴무 지정」을 누르는 즉시 관리자 캘린더에 자동 반영됩니다. 별도 저장 버튼을 누를 필요가 없습니다.

⚠ 이미 배정된 날짜도 확인하세요
배정 건이 잡혀 있는 날을 휴무로 설정해도 기존 배정이 자동 취소되지 않습니다. 먼저 관리자에게 알린 후 설정하세요.
💡 정기 휴무 미리 등록
매주 같은 요일이 쉬는 날이라면 다음 달치를 미리 한꺼번에 설정해 두면, 관리자가 처음부터 그 날짜를 피해 배정합니다.

사용법 정리
C/S, 메시지, 전자계약 메뉴의 사용 방법을 한 페이지에 정리했습니다.
C/S 메뉴
  • 배정 건과 연결된 고객 클레임이 표시됩니다. 관리자가 나에게 전달한 C/S도 여기에 나타납니다.
  • C/S를 눌러 내용을 확인한 후, 어떻게 처리했는지 처리 방법을 입력하고 완료 처리합니다.
  • 아무 C/S가 없으면 「표시할 C/S가 없습니다」 메시지가 보입니다. 이것은 정상입니다.
메시지 메뉴
  • 운영팀과 1:1로 채팅하는 공간입니다. 카카오톡처럼 메시지를 주고받을 수 있습니다.
  • 하단 입력창에 내용을 쓰고 전송 버튼(▶)을 누르면 운영팀에 바로 전달됩니다.
  • 운영팀의 답장은 메시지 아이콘에 읽지 않은 수가 배지로 표시됩니다.
  • 현장 문제·긴급 연락·일정 조율 등 업무 연락은 이 메시지 창을 이용해 주세요.
전자계약 — 팀장 전용
  • 관리자가 발급한 계약서를 확인하고 서명할 수 있는 팀장 전용 메뉴입니다.
  • 목록에서 「미리보기」를 눌러 계약서를 먼저 읽고, 「PDF로 저장」으로 파일을 내려받을 수 있습니다.
  • 내용 확인 후 서명을 진행하면 완료됩니다. 체결은 보안 확인 페이지를 거칩니다.
💡 이름 버튼 — 내 정보 확인 및 로그아웃
내비게이션 바 오른쪽 끝 이름(예: cbiseo▾)을 누르면 내 프로필 정보와 로그아웃 버튼이 나타납니다.
