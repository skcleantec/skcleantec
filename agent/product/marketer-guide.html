청소비서 관리자 앱 사용설명서 01 / 38 — 앱 전체 구조 & 권한 레벨
관리자 앱은 PC 웹 브라우저에서 사용합니다. 상단 메뉴(GNB)는 역할에 따라 다르게 표시되며, 마케터는 6개 메뉴를 기본으로 사용합니다.
📊
대시보드
오늘 매출·접수 현황, 팀장별 랭킹, 운영 시간대 분석, 접수 변경 이력을 한눈에 봅니다.
📋
서비스접수
접수 목록·수정, 발주서 발급·목록, 견적서, C/S 관리, 페이백·리뷰까지 모든 접수 업무를 처리합니다.
📅
스케쥴
월별 달력에서 팀장별 배정 현황을 AM/PM 슬롯으로 확인합니다. 권역 필터로 특정 지역만 볼 수 있습니다.
🗂️
정보공유
DB 마켓플레이스. 업체 간 접수 정보를 장바구니·구매·판매 형태로 주고받습니다.
💰
광고비
마케터 핵심 메뉴. 광고비 입력(작업 종료), 기간별 성과 지표(ROAS·건당비용), 사용자별 집계를 관리합니다.
💬
메시지
팀장과 1:1 채팅, 전체 팀장에게 현장 공지를 일괄 발송합니다.
📞
텔레CRM
(기능 ON 시) PC 새 창에서 접수·스크립트·가격 3단 상담. 전화 TM 전용.
⚙️ 관리자 전용 GNB는 마케터에게 관리자 메뉴 권한(admin.*)이 하나라도 있을 때만 표시됩니다. 상단·사이드·하위 메뉴도 항목별 권한에 맞게 자동으로 숨겨집니다.

마케터 권한 — 3단계 프리셋 + 항목별 조정

📋
없음 (기본 업무)
접수·발주·C/S·부재·광고비 입력·정보공유 등 일상 업무. 배정·삭제·관리자 메뉴는 없음.
✏️
일부 (운영)
+ 팀장 배정·접수 삭제·담당 마케터 변경·엑셀 일괄 등. 관리자 전용 GNB는 숨김.
🔑
전체 (관리자 메뉴)
+ 사용자·정산·설정 등 관리자 메뉴. 체크리스트로 항목별 세부 조정 가능.
청소비서 관리자 앱 사용설명서 02 / 38 — 대시보드 ① 실시간 현황
대시보드 ①
실시간 운영 현황 한눈에
대시보드 최상단에는 업체 구독 정보와 마케팅 세션 상태가 표시됩니다. 광고비 정산의 시작점인 「작업 종료 및 정산」 버튼이 이 영역에 있습니다.
1
계정 및 서비스 이용 현황

현재 구독 플랜(Premium 등), 상태(정상/만료), 활성 계정 수·브랜드 수를 확인합니다.

2
텔레마케팅·광고비 블록

마케터가 하루 업무를 시작하면 세션이 「진행 중」 상태로 표시됩니다. 업무 종료 시 반드시 「🔴 작업 종료 및 정산」 버튼을 눌러 광고비를 입력해야 합니다.

  • 이 버튼을 눌러야 오늘 사용한 광고비(채널별 금액)를 입력할 수 있습니다
  • 입력하지 않으면 광고비 집계에 누락이 생깁니다
3
운영 시간대(KST) 차트

시간대별로 어떤 활동이 얼마나 일어났는지 막대 그래프로 보여줍니다. 7일 / 30일 / 90일 탭으로 기간을 바꿀 수 있습니다.

  • 발주서 발급 — 고객에게 발주서 링크를 보낸 건수
  • 부재 유입 — 고객이 부재 중이었던 유입 건수
  • 발주서 재출 — 고객이 발주서를 다시 제출한 건수
  • 접수 전환 — 발주서 제출 후 정식 접수로 전환된 건수
  • 보유 유입 / 예약 확정 — 내부 보유 고객 및 최종 예약 확정 건수
💡 가장 중요한 습관
하루 업무가 끝나면 대시보드로 돌아와 「작업 종료 및 정산」 버튼을 눌러 광고비를 입력하세요. 이 데이터가 광고비 페이지의 모든 지표(ROAS·건당비용 등)의 기초가 됩니다.
청소비서 관리자 앱 사용설명서 03 / 38 — 대시보드 ② 매출·분석·변경이력
대시보드 ②
매출·분석·변경이력
스크롤을 내리면 오늘/이번달 매출, 팀장별 실적, 지역별 접수 지도, 접수 변경 이력이 순서대로 나타납니다.
1
오늘·이번달 미분배·해피콜 미완

상단 3개 카운터로 당일 긴급 확인 사항을 빠르게 파악합니다.

  • 이번달 미분배 — 아직 팀장에게 배정하지 않은 접수 건수. 숫자를 클릭하면 해당 목록으로 이동
  • 해피콜 미완 — 작업일 전날까지 고객에게 전화 안내를 완료하지 못한 건수
2
매출 및 정산 현황

오늘 매출, 이번달 누적 매출, 최근 7일 매출 추이 차트, 팀장별 매출 비교를 확인합니다.

3
이번달 현장 업무 랭킹

팀장별 누적 이동거리(km)와 배정 건수 랭킹을 나란히 보여줍니다. 현장 업무 부담이 고르게 분배되고 있는지 점검할 수 있습니다.

4
접수·예약 분석 (지역별 지도)

지역별로 접수가 얼마나 들어오는지 지도 위에 시각화됩니다. 어느 지역에서 많이 오는지 파악해 마케팅 전략에 활용하세요.

5
접수 변경 이력

가장 최근에 수정·생성된 접수 건들의 변경 내용이 타임라인으로 기록됩니다. 누가, 언제, 무엇을 바꿨는지 추적할 수 있습니다.

청소비서 관리자 앱 사용설명서 04 / 38 — 서비스접수 · 접수 목록 조회
서비스접수
접수 목록 — 필터·조회
상단 메뉴 「서비스접수」를 누르면 접수 목록이 나타납니다. 다양한 필터로 원하는 접수만 빠르게 찾을 수 있습니다.
1
날짜 기간 탭

오늘 / 이번달 / 지난달 / 기간 직접 선택. 기본값은 이번달입니다.

2
주요 필터 항목
  • 마케터 확정예약 — 마케터별로 확정된 예약 건수를 집계해 드롭다운으로 선택
  • 접수자 — 접수를 등록한 직원 기준 필터
  • 팀장·타업체·현장검수·브랜드 — 배정 팀장, 협력업체, 현장검수 여부, 영업 브랜드로 좁히기
  • 상태 — 접수·예약확정·배정완료·진행중·완료·취소·삭제 등
3
접수 행 클릭 → 상세 수정 팝업

목록에서 접수 행을 클릭하면 수정 팝업이 열립니다. 팝업에서 고객 정보부터 결제까지 모든 내용을 수정할 수 있습니다. (다음 슬라이드 참조)

4
행 우측 버튼들
  • 상세 — 팝업으로 수정
  • 클레임 — 클레임 등록
  • 취소 / 삭제 — 접수 취소·삭제 처리
💡 마케터는 자신이 담당한 접수만 보임
마케터는 기본적으로 본인이 접수한 건만 보입니다. 권한이 「일부」 이상이면 다른 직원의 접수도 조회·수정할 수 있습니다.
청소비서 관리자 앱 사용설명서 05 / 38 — 접수 수정 팝업 · 6개 섹션
서비스접수
접수 수정 팝업 — 6개 섹션
접수 행을 클릭하면 수정 팝업이 열립니다. 위에서 아래로 6개 섹션이 펼쳐지며, 각 항목을 바로 수정·저장할 수 있습니다.
① 고객 정보·주소
고객명, 연락처, 주소, 동·층·비밀번호, 특이사항
② 유형·면적·방·주방
서비스 종류, 평수, 방·화장실·주방 수, 건물 유형(아파트·빌라 등)
③ 일정
예약일, 시간대(오전·오후·저녁), 상세 시간, 팀장 배정
④ 정산·옵션
기본금, 추가 옵션 항목·금액, 전문시공옵션 리뷰 배지
⑤ 결제금액 내역
총 결제금액, 결제수단, 할인·쿠폰 적용 내역
⑥ 메모·내부정보
내부용 메모, 마케터 지정, 브랜드 선택
💡 저장 방법
각 섹션에서 수정 후 해당 섹션 하단의 「저장」 버튼을 누릅니다. 팝업 전체를 한꺼번에 저장하는 것이 아니라 섹션별로 저장합니다.
청소비서 관리자 앱 사용설명서 06 / 38 — 발주서 발급
서비스접수 › 발주서
발주서 발급 — 고객 링크 생성
발주서는 고객이 서비스 내용을 직접 확인하고 제출하는 온라인 폼입니다. 마케터가 발주서를 만들어 고객에게 링크를 보내면, 고객이 폼을 작성·제출합니다.
1
양식 선택

발주서 양식을 선택합니다. 업체에서 미리 만들어 둔 여러 가지 양식 중 고객 유형에 맞는 것을 고릅니다.

2
대기접수 연결 (선택)

이미 대기 중인 접수가 있다면 연결합니다. 연결하면 고객이 발주서를 제출할 때 기존 접수와 자동으로 매칭됩니다.

3
내부표시 이모지

이 발주서를 식별하기 위한 이모지를 선택합니다. 목록에서 쉽게 구분할 수 있습니다.

4
고객 정보 입력

고객명, 연락처, 주소를 입력합니다. 기존 접수와 연결한 경우 자동으로 채워집니다.

5
발급 금액 설정 후 발송

발주서에 표시될 금액을 입력합니다. 완료 후 링크 복사, 카카오 발송, 문자 발송 중 선택합니다.

⚠️ 발주서 vs 접수의 차이
발주서는 고객에게 보내는 견적·확인 폼입니다. 고객이 제출하면 그때 정식 「접수」로 전환됩니다. 발주서만 발급하고 고객이 제출하지 않으면 접수 목록에 나타나지 않습니다.
청소비서 관리자 앱 사용설명서 07 / 38 — 발주서 목록
서비스접수 › 발주서 목록
발주서 목록 — 제출 현황 관리
지금까지 발급한 발주서를 한눈에 확인합니다. 고객이 제출했는지 여부를 추적하고, 확인 이메일을 재발송할 수 있습니다.
1
기간 탭 선택

C/S 관리 — 고객 문의 처리
서비스를 이용한 고객의 사후 문의·불만을 관리합니다. 고객에게 C/S 페이지 링크를 공유하면 고객이 직접 문의를 입력할 수 있습니다.
1
C/S 링크 복사 후 공유

목록에서 해당 접수의 「링크복사」를 클릭하면 고객용 C/S 페이지 URL이 복사됩니다. 문자·카카오로 고객에게 보내세요.

2
고객 문의 확인

고객이 C/S 페이지에서 문의를 남기면 해당 접수의 상태가 변경됩니다. 문의 내용을 확인하고 처리합니다.

3
기간 필터

접수일 기준으로 최근 3개월 / 월별 / 날짜 직접 지정으로 조회 범위를 조정합니다.

4
C/S 페이지 미리보기

고객이 실제로 보게 될 C/S 페이지를 미리 확인할 수 있습니다. 페이지 우측 「미리보기」 버튼을 클릭하세요.

페이백·리뷰 — 리뷰 보상 관리
청소 완료 후 고객이 리뷰를 작성하면 페이백(현금 보상)을 신청할 수 있습니다. 이 화면에서 신청을 검토하고 입금 처리합니다.
1
탭으로 단계별 관리
  • 전체 — 모든 페이백 신청 내역
  • 신청접수 — 고객이 방금 신청한 건. 처리 대기 상태
  • 리뷰확인 — 리뷰 작성 여부를 담당자가 확인 중인 상태
  • 입금완료 — 페이백 금액을 고객 계좌로 입금 처리 완료
  • 반려 — 리뷰 조건 미충족·허위 등으로 거절한 건
  • 미확인만 — 아직 아무도 확인하지 않은 신청만 표시
2
처리 순서

신청접수 → 리뷰 링크 클릭하여 실제 리뷰 확인 → 「리뷰 확인」 클릭 → 계좌번호 확인 후 입금 → 「입금 완료」 클릭

3
반려 처리

리뷰가 조건에 맞지 않거나 허위인 경우 「반려」 버튼을 눌러 처리합니다. 반려 사유를 입력할 수 있습니다.

광고비 ② — 사용자별 집계
기간 요약 카드 아래에 마케터별 성과 테이블이 나타납니다. 각 마케터가 얼마의 광고비를 써서 얼마의 성과를 냈는지 한 눈에 비교합니다.
1
테이블 컬럼 설명
  • 이름·역할 — 마케터 이름과 역할 배지
  • 광고비 — 해당 마케터가 해당 기간에 입력한 총 광고비
  • 예약완료 — 예약으로 전환된 건수
  • 미제출·취소·삭제 — 각각의 건수 (낮을수록 좋음)
  • 매출 — 예약완료 건들의 총 결제금액
  • ROAS — 마케터 개인의 광고 효율
  • 건당 비용 — 1건 예약당 평균 광고비
  • 일평균 광고비 — 선택 기간 동안 하루 평균 광고비
2
「일별」 버튼 → 일별 정산 팝업

각 마케터 행 우측 「일별」 버튼을 누르면 날짜별 광고비 내역 팝업이 열립니다. 어떤 날 어떤 채널에 얼마를 썼는지 확인할 수 있습니다.

3

광고비 입력은 대시보드의 「작업 종료 및 정산」 버튼에서 시작합니다. 입력 완료된 내역은 광고비 페이지 하단 작업 종료 이력에 쌓입니다.

광고비 입력 방법 (대시보드에서 시작)

  • 과목(항목) 단위로 건수와 금액을 입력할 수 있습니다
  • 완료 후 「종료」를 누르면 정산 기록이 저장됩니다
2
작업 종료 이력 확인

광고비 페이지 하단에서 날짜별·마케터별 입력 이력을 확인합니다.

  • 담당 — 해당 마케터 이름
  • 채널별 금액 — 채널별로 입력한 금액과 과목 건수 내역
3
하루에 여러 번 입력 가능

오전·오후에 나눠서 입력하거나 실수 수정을 위해 같은 날 여러 번 작업 종료 버튼을 눌러도 됩니다. 이력은 시각순으로 모두 기록됩니다.

광고비 › 설정
광고비 설정 — 채널·과목 관리
광고비 메뉴 좌측 「설정」을 누르면 광고 채널을 추가·수정·삭제하고 과목(건당 금액)을 관리할 수 있습니다.
1
광고 채널 추가

하단 「+ 채널 추가」를 눌러 새 광고 채널(예: 당근, 카카오 등)을 등록합니다. 채널 이름을 입력하고 저장합니다.

2
정산 방식 선택
  • 건당 금액 — 과목별로 건수를 입력하면 금액이 자동 계산됩니다. (예: 전화상담 1건 = 5,000원)
  • 직접 입력 — 매번 금액을 직접 입력합니다. 정산 방식이 복잡하거나 고정되지 않은 채널에 적합합니다.
3
과목(항목) 관리

건당 금액 방식을 사용하는 채널에는 과목을 추가합니다. 과목명과 건당 금액을 입력하면 작업 종료 시 건수만 입력해도 금액이 자동으로 계산됩니다.

4
채널 사용 ON/OFF

일시적으로 사용하지 않는 채널은 삭제하지 않고 비활성화만 해둘 수 있습니다. 비활성화된 채널은 작업 종료 팝업에 표시되지 않습니다.

  • 인원조정 · 팀장조정 · 배정현황 · 가용인원
  • 일정마감 / 일정마감 해제 — 전체·오전만·오후만 (비밀번호 확인)
  • 지도 — 그날 접수 주소를 지도로 보기

직원권한 화면 (스크린샷 교체 가능)

관리자 전용 › 설정
직원권한 — 마케터 항목별 권한
ADMIN 전용. 마케터마다 3단계 프리셋으로 빠르게 채운 뒤, 도메인별 체크리스트로 세부 조정합니다. 저장 시 그 맵이 실제 메뉴·API 권한이 됩니다.
1
마케터 선택

왼쪽 목록 또는 URL ?userId= 로 대상 선택. 사용자 등록 모달의 「항목별 상세 권한 설정 →」 링크로도 바로 이동할 수 있습니다.

2
3단계 라디오
  • 없음 — 접수·발주·C/S·광고비 등 기본 업무
  • 일부 — + 배정·삭제·고급 수정 (관리자 GNB 숨김)
  • 전체 — + 사용자·정산·설정 등 관리자 메뉴

프리셋 변경·되돌리기 시 확인 창이 뜹니다. 체크를 수동으로 바꾸면 「일부 수정됨」 표시.

3
도메인별 체크 · 저장

접수·스케줄·발주·견적·C/S·광고·정보공유·관리자 메뉴 등 그룹별로 ON/OFF. 권한 저장 후 GNB·사이드·버튼·API에 즉시 반영됩니다.

잠금 항목(직원권한 화면 등)은 마케터에게 부여할 수 없습니다.

GNB › 텔레CRM
텔레CRM — 전화 상담 전용 3단 화면
PC에서만 사용합니다. GNB 「텔레CRM」을 누르면 새 창이 열리며, 왼쪽 접수·가운데 스크립트·오른쪽 가격을 한 화면에서 보며 상담합니다. 업체 기능 mod_telecrm이 켜져 있어야 메뉴가 보입니다.
1
접수 · 고객 (왼쪽)
  • 신규 — 연락처 입력 후 부재·보류·예약금·예약완료 등 처리 구분 선택 → 저장
  • 기존 — 연락처로 이력 조회 → 이름·주소·평수 자동 채움
  • 접수 이력 행 클릭 → 접수 상세 수정 모달 (서비스접수와 동일)
  • 발주서 발급 화면 열기 — 새 탭에서 발주서 발급
2
스크립트 (가운데)

관리자가 등록한 카테고리·탭을 읽으며 상담합니다. {고객명} {평수} {예상가} 가 접수·가격과 연동되어 치환됩니다. 탭 내용은 클릭 한 번으로 복사할 수 있습니다.

3
가격 (오른쪽)

평수·카테고리별 단가표로 견적을 안내합니다. 평수는 접수 폼 「추가 필드」·가격 패널·스크립트에 동일하게 연동됩니다. 항목 금액·예상가를 복사해 문자·카톡에 붙여 넣을 수 있습니다.

4
광고 세션 · 설정
  • 헤더 광고 세션 — 대시보드와 동일하게 세션 종료·정산 (권한·mod_advertising 필요)
  • 설정 — 스크립트·가격 카테고리·기본 평당 단가 (crm.settings 권한)
💡 권한
crm.view — 텔레CRM 메뉴·작업 화면 · crm.settings — 설정 CRUD · 접수 저장은 inquiry.create 등 기존 접수 권한을 따릅니다.
