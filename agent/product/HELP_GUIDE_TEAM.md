# 팀장 앱 사용 설명서 (청소비서)

> 이 문서는 청소비서 팀장 앱의 각 메뉴와 기능을 단계별로 설명합니다.  
> team-guide.html 인포그래픽과 동일한 내용을 텍스트 형식으로 정리한 참조 문서입니다.  
> 최종 수정: 2026년 6월

---

## 전체 처리 순서 — 신규 배정이 왔을 때 (10단계)

새 배정 알림을 받은 후 완료까지 팀장이 따라야 할 순서입니다.

| 단계 | 행동 | 상세 |
|------|------|------|
| 1 | 배정 알림 확인 | 내비게이션 `배정` 배지 숫자 확인. 빨간 태그(신규배정) = 확인 전 건 |
| 2 | 접수 상세 열기 | 카드 누르면 팝업 오픈. 위→아래 전체 읽기. 제목·특이사항 필수 확인 |
| 3 | 고객·현장 확인 | 주소, 방화 구조, 건물 유형, 평수, 특이사항 메모 확인 |
| 4 | 해피콜 전화 | 고객 연락처 눌러 전화 → 일정 안내 → 「해피콜 완료」 버튼 누르기. 마감: 작업일 전날 |
| 5 | 팀원 확인 | 투입 팀원 목록·연락처 확인. 파란 번호 누르면 바로 전화 |
| 6 | 미팅 시간 설정 | 팝업 「미팅시간」에 집합 시간 입력 → 저장. 팀원 크루 앱에 자동 공유 |
| 7 | 청소 전 촬영 | 현장 도착 후, 작업 전 파란 「청소 전 촬영」 버튼 눌러 사진 등록 |
| 8 | 청소 진행 | 청소 작업. 추가 비용 발생 시 결제금액 → 추가결재 「펼치기」에 항목 등록 |
| 9 | 청소 후 촬영 | 완료 후 초록 「청소 후 촬영」 버튼 눌러 사진 등록 |
| 10 | 완료 처리 + 결제 | 검정 「현장 검수·청소완료」 버튼 → 상태 자동 완료 전환. 카드결제 필요 시 처리 |

## 1. 대시보드 — `/team/dashboard`

**화면 목표**  
앱 진입 시 가장 먼저 보이는 홈 화면. 오늘 배정 일정, 해피콜 알림, 전체 통계를 한눈에 확인합니다.

**주요 구성 요소**

| 요소 | 설명 |
|------|------|
| **해피콜 알림 박스** | 빨간 박스. 마감 초과 N건 / 마감 전 미완 N건. 숫자 누르면 해당 목록 이동 |
| **8칸 통계 카드** | 전체·오늘·분배·진행·완료·접수·보류·취소 건수 |
| **오늘 일정 카드** | 고객명, 관리자 메모, 시간, 방화 구조, 주소, 상태 배지 |
| **전화 버튼** | 카드 오른쪽 파란 버튼. 누르면 고객에게 즉시 전화 |
| **알림 배지** | 우상단. 읽지 않은 변경 이력·공지 수 |

**일정 카드 태그 색상**

- **빨간 태그** (오늘·신규배정) — 오늘 작업인데 아직 확인 전
- **노란 태그** (변경이력있음) — 날짜·금액·주소 중 하나 이상 변경됨. 반드시 상세 확인
- **태그 없음** — 일반 건

**통계 카드 의미**

- `전체` — 나에게 배정된 모든 건
- `오늘` — 오늘 작업 예정
- `분배` — 팀원 배치 완료
- `진행` — 현장 청소 중
- `완료` — 청소 완료 처리됨
- `접수` — 신규 접수, 팀원 배치 전
- `보류` — 확인 중·미루기
- `취소` — 고객 요청 등 취소

**해피콜 처리 흐름**  
빨간 박스 숫자 누르기 → 카드 열기 → 고객 전화 → 「해피콜 완료」 버튼 누르기

## 2. 배정목록 — `/team/assignments`

**화면 목표**  
나에게 배정된 모든 건(과거·현재·미래)을 검색·필터링합니다.

**주요 구성 요소**

| 요소 | 설명 |
|------|------|
| **기간 기준 드롭다운** | 배정일(본인) / 접수일 / 예약일 중 선택 |
| **기간 탭** | 당일 / 전체 / 월별(연도·월 선택) / 날짜(시작~종료 직접 입력) |
| **상태 드롭다운** | 전체·대기·접수·분배·진행·완료·보류·취소·C/S |
| **검색창** | 고객명·연락처·주소·접수번호로 검색 후 조회 버튼 |
| **현장 카드** | 이름·접수번호·배정일·담당자·주소·팀원·상태 |

**실전 검색 조합**

- 오늘 현장만: 탭 `당일`
- 이번 달 완료 건: 탭 `월별` + 상태 `완료`
- 특정 고객: 이름 검색 → 조회

## 3. 배정 상세 팝업 — 섹션 구조

카드 누르면 팝업이 열립니다. 위에서 아래로 6개 구역으로 나뉩니다.

### 상단 섹션 (스크롤 없이 보임)

| 섹션 | 내용 |
|------|------|
| **헤더** | 고객명, SK로 시작하는 접수번호, 상태 배지 |
| **담당 팀장** | 내 이름 옆 📞 = 내 등록 연락처 확인 |
| **접수 정보** | 접수일, 담당 마케터 + 전화걸기 버튼 |
| **제목** | 관리자 메모 ("오늘·신규배정", "4인투입", "사이청소" 등). 반드시 읽기 |
| **배정 정보** | 배정일시, 배정자, 공동 배정 여부 |
| **고객·현장** | 연락처(파란색 → 누르면 전화), 주소, 평수, 방화, 건축물 유형 |

### 중간 섹션 (스크롤 내리면 나옴)

| 섹션 | 내용 |
|------|------|
| **투입 팀원** | 팀원 이름·연락처. 파란 번호 누르면 바로 전화 |
| **미팅시간** | 집합 시간 입력 → 저장 → 팀원 크루 앱에 자동 공유 |
| **일정** | 예약일, 희망 시간. 「접수 변경 이력 보기」로 변경 내역 확인. 「예약일 변경」으로 수정 후 관리자 화면 즉시 반영 |
| **특이사항** | 관리자·팀장·타업체 공유 메모. 노란 배경. 직접 추가 가능 |

### 하단 섹션 (더 스크롤 내리면 나옴)

| 섹션 | 내용 |
|------|------|
| **결제 금액 내역** | 총 결제금액 / 예약금(선결제) / 잔금(현장에서 받을 금액) |
| **추가결재** | 현장 추가비용 등록 (하단 상세 참조) |
| **해피콜** | 미완료 시 빨간 경고 표시 |
| **상담·참고 / 썸네일** | 마케터 메모, 미리보기 사진 |
| **현장 사진** | 청소 전·후 촬영 사진 보관 |
| **액션 버튼** | 청소 전 촬영 / 청소 후 촬영 / 현장 검수·완료 / 해피콜 완료 / 카드결제 / 닫기 |

**액션 버튼 순서**

1. 📷 **청소 전 촬영** (파란색) — 작업 전 현장 상태
2. 📞 **해피콜 완료** (하늘색) — 고객 전화 후 처리
3. 📷 **청소 후 촬영** (초록색) — 청소 결과물
4. ✅ **현장 검수·청소완료** (검정) — 상태 즉시 완료 전환
5. 💳 **카드결제** (초록 테두리) — 잔금 현장 카드 수납

## 4. 추가결재 — 현장 추가비용 처리

**위치**: 배정 상세 팝업 → 결제 금액 내역 섹션 → 「추가결재 (별도 정산) 펼치기」

**결제 금액 구조**

- **총 결제금액**: 계약 원금. 변경 안 됨.
- **예약금**: 이미 선결제된 금액.
- **잔금**: 현장에서 받을 금액 = 기본 잔금 + 회사입금 추가결재 합계.

**추가결재 입력 방법**

1. 「추가결재 (별도 정산)」 항목 눌러 펼치기
2. 항목명 입력 (예: "발코니 특수청소", "에어컨 청소 추가")
3. 금액 입력. 빠른 버튼 (+1천 / +5천 / +1만 / +10만) 활용 가능
4. **수금 방식 선택 (필수)** — 선택하지 않으면 저장 안 됨
5. 저장 버튼 누르기

**수금 방식 2가지**

| 방식 | 설명 | 잔금 반영 | 급여 정산 |
|------|------|-----------|-----------|
| **현장결재 (본인이 받음)** | 팀장이 현장에서 직접 현금·계좌이체로 수령 | ❌ 포함 안 됨 (이미 받은 금액) | ✅ 포함 |
| **회사입금** | 고객이 나중에 회사 계좌로 입금 | ✅ 잔금에 추가됨 | ✅ 포함 |

**저장 후 자동 처리**

- 접수 변경 이력에 자동 기록 ("추가 청소/비용 추가: [항목명] +[금액]원")
- 관리자 화면에 실시간 반영
- 잘못 입력 시 항목 옆 **수정** / **삭제** 버튼으로 수정 가능

> ⚠ **주의**: 현장에서 직접 받은 돈은 반드시 `현장결재`로 선택하세요.  
> `회사입금`으로 잘못 선택하면 잔금에 중복 포함되어 관리자 정산이 어긋납니다.

## 5. 청소 전 촬영 — `/team/pre-clean/:id`

**화면 목표**  
청소 시작 전 현장 상태를 구역별로 사진 촬영합니다. 고객 분쟁 시 Before 증거가 됩니다.

**진입 방법**  
배정 상세 팝업 하단 파란 「📷 청소 전 촬영」 버튼

**주요 구성 요소**

| 요소 | 설명 |
|------|------|
| **진행 표시** | 「진행: 청소 전 N/M 항목」. 완료 시 「— 청소 전 촬영 완료」로 변경 |
| **구역 목록** | 주방·안방·작은방·욕실·베란다 등. 구역마다 완료/진행중/대기 상태 표시 |
| **항목별 카메라** | 구역 누르면 세부 항목과 카메라 버튼 표시. 항목마다 사진 1장 등록 |
| **「해당없음」** | 해당 공간이 없거나 청소 범위 외면 선택. 사진 없이 완료 처리됨 |
| **별표(★) 버튼** | 오염이 심한 항목 사진에 별표 표시. 관리자·고객에게 주의 항목으로 전달 |
| **하단 이동 버튼** | 「현장 검수 · 청소완료로 이동」 → 검수 화면으로 이동 |

**사용 방법**

1. 「📷 청소 전 촬영」 버튼 누르기
2. 첫 번째 구역부터 순서대로 항목별 사진 촬영
3. 해당 없는 공간은 「해당없음」 선택
4. 오염이 심한 곳은 별표(★) 눌러 추가 기록
5. 전체 완료 후 「현장 검수 · 청소완료로 이동」 버튼 누르기

> ⚠ 사진을 찍지 않아도 강제로 막히지 않습니다. 하지만 나중에 고객 클레임 시 사진이 없으면 팀장이 불리해집니다.

## 6. 청소 후 촬영 — `/team/post-clean/:id`

**화면 목표**  
청소 완료 후 같은 구역을 다시 촬영합니다. 고객에게 전달되는 After 증거입니다.

**진입 방법**
- 배정 상세 팝업 하단 초록 「📷 청소 후 촬영」 버튼
- 현장검수 화면 상단 「청소 후 촬영」 링크 버튼

**주요 구성 요소**

| 요소 | 설명 |
|------|------|
| **진행 표시** | 「진행: 청소 후 N/M 항목」. 완료 시 「— 청소 후 촬영 완료」로 변경 |
| **「청소 후 촬영 시작」 버튼** | 미완료 항목이 있을 때 초록 버튼 표시. 누르면 첫 번째 미완료 항목으로 이동 |
| **구역 목록** | 청소 전과 동일한 순서. 완료 구역은 초록 완료 표시 |
| **하단 이동 버튼** | 「현장 검수 · 청소완료로 이동」 |

**사용 방법**

1. 「📷 청소 후 촬영」 버튼 누르기
2. 「청소 후 촬영 시작」 버튼 누르기 → 첫 미완료 항목으로 자동 이동
3. 청소 전과 동일한 위치·각도로 After 사진 촬영
4. 「해당없음」 설정된 항목은 자동으로 건너뜀
5. 완료 후 「현장 검수 · 청소완료로 이동」 버튼 누르기

> ⚠ 사진 없이 완료 처리는 가능하지만, 나중에 고객 클레임 시 불리합니다. 꼭 찍어 두세요.

## 7. 현장검수 · 청소완료 — `/team/inspection/:id`

**화면 목표**  
청소가 끝나면 팀장과 고객이 함께 확인·서명하고 완료 처리합니다. **완료 후에는 수정 불가합니다.**

**진입 방법**
- 청소 전/후 촬영 화면 하단 「현장 검수 · 청소완료로 이동」 버튼
- 배정 상세 팝업 하단 검정 「✅ 현장 검수·청소완료」 버튼

**화면 구성 — 위에서 아래로**

| 섹션 | 내용 |
|------|------|
| **기본사항** | 4개 확인 질문. 팀장·고객이 각각 예/아니오 선택 |
| **구역별 사진** | 청소 전·후 사진이 구역별로 정리 표시. 빠진 사진 추가 가능 |
| **특이사항 (팀장)** | 현장 특이점, 고객 요청사항 등 자유 메모 |
| **고객 이메일** | 완료본(PDF) 수신 이메일 주소 입력 (필수) |
| **동의 항목** | 필수 5개 + 선택 1개. 「전체 동의」 버튼으로 한번에 체크 |
| **고객 서명** | 서명 박스에 고객이 직접 손가락으로 서명 → 「서명 저장」 |
| **완료 버튼** | 「청소완료 (고객 확인·서명)」 — 확인 창 후 최종 완료 처리 |

**기본사항 4개 질문 (팀장·고객 각각 답변)**

1. 서비스 전날 담당 팀장으로부터 연락을 받으셨습니까?
2. 추가 비용이 발생할 수 있는 사항에 대해 안내를 받으셨습니까?
3. 약속된 시간에 청소가 시작되었습니까?
4. 청소 중 발생한 쓰레기를 봉투에 담아 정리하였습니까?

**동의 항목 (필수 5개)**

- 개인정보 수집·이용 동의
- 개인정보 제3자 제공 동의
- 현장 검수 및 청소 범위 확인
- 현장검수 완료 및 기록의 정확성 (팀장)
- 현장검수 완료 및 기록의 정확성 (고객)

**완료 처리 순서**

1. 기본사항 4개 질문 — 팀장 먼저, 고객 순으로 예/아니오 선택
2. 고객 이메일 주소 입력 (고객과 함께 확인)
3. 「전체 동의」 버튼 → 고객이 동의 항목 확인 후 체크
4. 고객이 서명 박스에 직접 서명 → 「서명 저장」
5. 「청소완료 (고객 확인·서명)」 버튼 → 확인 창 → 완료

**완료 후 자동 처리**

- 접수 상태가 「완료」로 변경
- 고객 이메일로 완료본 PDF 자동 발송
- 「고객 열람 링크 복사」 버튼 활성화 → 링크를 고객에게 문자로 전송 가능

> ⚠ **완료 후에는 수정이 불가합니다.** 내용을 바꾸려면 관리자가 직접 무효 처리 후 재작성해야 합니다. 이메일·서명·동의 항목을 반드시 고객과 함께 확인한 뒤 버튼을 누르세요.

## 8. 스케줄 달력 — `/team/schedule`

**화면 목표**  
월간 달력으로 배정 일정을 시각적으로 확인하고, 날짜 클릭으로 그날 상세 카드 목록을 봅니다.

**주요 구성 요소**

| 요소 | 설명 |
|------|------|
| **연도/월 드롭다운** | 다른 달 이동 가능. 다음 달도 미리 확인 가능 |
| **달력 그리드** | 날짜 아래 청록 숫자 = 그날 배정 건수 |
| **청록색 날짜 숫자** | 손없는날 (음력 끝자리 9·0). 수요 높은 날 |
| **파란 원 날짜** | 오늘 날짜 표시 |

**날짜 클릭 → 상세 카드 목록**

- 달력 아래에 그날의 배정 건 카드 목록이 바로 펼쳐짐
- 각 카드에 고객명·연락처·주소·팀원·시간·상태 배지 표시
- 카드 오른쪽 전화 버튼 → 고객 바로 전화
- 카드 누르기 → 상세 팝업 열기

**스케줄 vs 배정목록 차이**

- **스케줄** → 달력 뷰. 어떤 날이 바쁜지 한눈에 파악
- **배정목록** → 목록 뷰. 검색·필터로 특정 건 빠르게 찾기

## 9. 휴무일 설정 — `/team/dayoffs`

**화면 목표**  
쉬는 날을 직접 달력에서 지정합니다. 지정 즉시 관리자 캘린더에 반영됩니다.

**사용 방법**

1. 연도·월 드롭다운에서 원하는 달로 이동
2. 쉬고 싶은 날짜 누르기 → 확인 창 → 「휴무 지정」
3. 빨간 원으로 표시됨. 별도 저장 버튼 불필요 (즉시 반영)
4. 이미 빨간 날짜 다시 누르기 → 「취소 확인」 → 일반 날짜로 복원

> ⚠ **주의**: 배정 건이 잡혀 있는 날을 휴무로 설정해도 기존 배정이 자동 취소되지 않습니다. 먼저 관리자에게 알리세요.

> 💡 **팁**: 정기 휴무는 다음 달치를 미리 한꺼번에 설정해 두면 관리자가 처음부터 그 날짜를 피해 배정합니다.

## 10. C/S (고객 문의) — `/team/cs`

**화면 목표**  
관리자가 전달한 고객 클레임·AS 요청을 확인하고 처리 내용을 기록합니다.

**사용 방법**

1. C/S 항목 누르기 → 내용 확인
2. 어떻게 처리했는지 처리 방법 입력
3. 완료 처리 버튼 누르기
4. 표시할 C/S가 없으면 「표시할 C/S가 없습니다」 — 정상 상태

## 11. 메시지 — `/team/messages`

**화면 목표**  
운영팀(관리자)과 1:1 채팅으로 업무 연락을 주고받습니다.

**사용 방법**

1. 하단 입력창에 메시지 입력
2. 전송 버튼(▶) 누르기
3. 운영팀의 답장은 내비게이션 메시지 아이콘 배지로 표시됨
4. 현장 문제·긴급 연락·일정 조율은 이 창 사용

---

## 12. 전자계약 — `/team/e-contracts`

**화면 목표**  
관리자가 발급한 계약서를 확인하고 전자서명합니다. 팀장 전용 메뉴.

**사용 방법**

1. 목록에서 미서명 계약 확인
2. 「미리보기」로 계약서 내용 먼저 읽기
3. 「PDF로 저장」으로 파일 다운로드 가능
4. 내용 확인 후 서명 진행 → 보안 확인 페이지 통과 → 완료

## 앱 사용 팁

1. **모바일 최적화**: 팀장 화면은 스마트폰·태블릿에서 사용하도록 설계됐습니다
2. **내 이름 버튼**: 내비게이션 오른쪽 끝 이름(▾) 누르면 내 프로필·로그아웃
3. **실시간 반영**: 미팅시간 저장, 해피콜 완료, 완료 처리는 즉시 관리자 화면에 반영됨
4. **추가결재 수금 방식**: 현장에서 직접 받으면 반드시 `현장결재` 선택
5. **변경이력 태그**: 노란 태그가 붙은 건은 반드시 상세에서 변경 내용 확인
