## 1. 앱 전체 구조 & 권한 레벨

관리자 앱은 **PC 웹 브라우저**에서 사용합니다 (www.cbiseo.com → 업체코드 입력 → 로그인).

### 상단 메뉴(GNB) 6개

| 메뉴 | 주요 기능 |
|------|-----------|
| 대시보드 | 매출·접수·운영 현황 실시간 확인 |
| 서비스접수 | 접수 목록·수정, 발주서·견적서·C/S·페이백 |
| 스케쥴 | 월별 달력, 팀장 배정 현황 |
| 정보공유 | DB 마켓플레이스 |
| 광고비 | 광고비 입력·분석·성과 지표 |
| 메시지 | 팀장 1:1 채팅, 현장 공지 일괄 발송 |
| **텔레CRM** | 전화 상담 3단 화면 (PC 팝업, `mod_telecrm`) |

> **관리자 전용** 메뉴(사용자 등록·정산·설정 등)는 권한 레벨 「전체」인 마케터와 관리자만 볼 수 있습니다.

### 마케터 권한 레벨 3단계

| 레벨 | 가능한 작업 |
|------|------------|
| **없음 (조회만)** | 목록·상세 조회만 가능. 수정·삭제 불가 |
| **일부 (운영 권한)** | 배정·삭제·접수 고급 수정 가능. 관리자 전용 메뉴 비표시 |
| **전체 (관리자 동일)** | 모든 기능 사용 가능. 관리자 전용 메뉴 포함 |

## 2. 대시보드 — 실시간 현황

**경로**: 상단 메뉴 「대시보드」

### 계정 및 서비스 이용 현황
- 현재 구독 플랜(Premium 등), 상태(정상/만료), 활성 계정 수 확인

### 텔레마케팅·광고비 블록 ⭐ 핵심
- 마케터가 업무 중이면 **「진행 중」** 상태로 표시됩니다
- 업무 종료 시 반드시 **「🔴 작업 종료 및 정산」** 버튼을 눌러 광고비를 입력해야 합니다
- 이 데이터가 광고비 페이지의 모든 지표(ROAS·건당비용 등)의 기초가 됩니다

### 운영 시간대(KST) 차트
- 시간대별 활동량을 막대 그래프로 표시
- 7일 / 30일 / 90일 탭으로 기간 변경
- 항목: 발주서 발급 / 부재 유입 / 발주서 재출 / 접수 전환 / 보유 유입 / 예약 확정

## 3. 대시보드 — 매출·분석·변경이력

스크롤을 내리면 나타나는 하단 섹션들입니다.

### 상단 3개 카운터
- **오늘 접수** — 당일 신규 접수 건수
- **이번달 미분배** — 팀장에게 아직 배정되지 않은 접수 (클릭 시 해당 목록으로 이동)
- **해피콜 미완** — 작업일 전날까지 안내 전화를 완료하지 못한 건수

### 매출 및 정산 현황
- 오늘 매출, 이번달 누적 매출
- 최근 7일 매출 추이 차트
- 팀장별 매출 랭킹 (누적 거리 순위 / 배정 건수 순위)

### 접수·예약 분석
- 지역별 접수 분포 지도

### 접수 변경 이력
- 최근 수정·생성된 접수 건들의 변경 타임라인
- 누가, 언제, 무엇을 바꿨는지 기록

## 4. 서비스접수 — 접수 목록 조회

**경로**: 상단 메뉴 「서비스접수」

### 날짜 기간 탭
오늘 / 이번달 / 지난달 / 기간 직접 선택

### 주요 필터
| 필터 | 설명 |
|------|------|
| 마케터 확정예약 | 마케터별 확정 예약 건수 집계 |
| 접수자 | 접수를 등록한 직원 기준 |
| 팀장 | 배정된 팀장 기준 |
| 타업체·현장검수·브랜드 | 협력업체, 현장검수 여부, 영업 브랜드 |
| 상태 | 접수·예약확정·배정완료·진행중·완료·취소·삭제 |

### 접수 행 클릭 → 수정 팝업
- 행을 클릭하면 수정 팝업이 열립니다
- 행 우측 버튼: **상세** / **클레임** / **취소** / **삭제**

> 💡 마케터는 기본적으로 본인이 접수한 건만 보입니다. 권한 「일부」 이상이면 타 직원 접수도 조회·수정 가능합니다.

## 5. 서비스접수 — 접수 수정 팝업

접수 행 클릭 시 열리는 팝업의 6개 섹션입니다.

| 섹션 | 포함 내용 |
|------|-----------|
| ① 고객 정보·주소 | 고객명, 연락처, 주소, 동·층·비밀번호, 특이사항 |
| ② 유형·면적·방·주방 | 서비스 종류, 평수, 방·화장실·주방 수, 건물 유형 |
| ③ 일정 | 예약일, 시간대(오전·오후·저녁), 상세 시간, 팀장 배정 |
| ④ 정산·옵션 | 기본금, 추가 옵션 항목·금액, 전문시공옵션 |
| ⑤ 결제금액 내역 | 총 결제금액, 결제수단, 할인·쿠폰 |
| ⑥ 메모·내부정보 | 내부용 메모, 마케터 지정, 브랜드 선택 |

> 💡 섹션별로 저장 버튼이 따로 있습니다. 한꺼번에 저장하는 것이 아니라 **섹션별로 저장**합니다.

## 6. 발주서 발급

**경로**: 서비스접수 → 사이드 메뉴 「발주서 발급」  
**URL**: `/admin/inquiries/order-issue`

발주서는 고객이 서비스 내용을 확인하고 제출하는 온라인 폼입니다.

### 발급 순서
1. **양식 선택** — 업체에서 미리 만들어 둔 양식 중 선택
2. **대기접수 연결** (선택) — 기존 대기 접수가 있으면 연결
3. **내부표시 이모지** — 목록에서 구분용 이모지 선택
4. **고객 정보 입력** — 고객명, 연락처, 주소
5. **발급 금액 설정**
6. **발송** — 링크 복사 / 카카오 발송 / 문자 발송 중 선택

> ⚠️ **발주서 vs 접수**: 발주서는 고객에게 보내는 확인 폼입니다. 고객이 제출해야 비로소 정식 「접수」로 전환됩니다.

### 주요 기능
- **담당자 필터** — 마케터별로 보기

## 8. 견적서

**경로**: 서비스접수 → 사이드 메뉴 「견적서」  
**URL**: `/admin/inquiries/quotations`

### 견적서 작성 순서
1. 우측 상단 **「+ 새 견적서」** 버튼 클릭
2. 상대방(고객) 정보, 서비스 항목·금액 입력
3. **「확정」** 상태로 변경
4. 고객 이메일로 **PDF 발송**

### 상태
| 상태 | 의미 |
|------|------|
| 작성 중 | 편집 중. 아직 고객에게 발송 전 |
| 확정 | 내용 확정 완료. 발송 준비 상태 |
| 발송됨 | 고객 이메일로 PDF 발송 완료 |

## 9. C/S 관리

**경로**: 서비스접수 → 사이드 메뉴 「C/S 관리」  
**URL**: `/admin/inquiries/cs`

서비스 완료 후 고객의 사후 문의·불만을 관리합니다.

### 사용 방법
1. 목록에서 해당 접수의 **「링크복사」** 클릭 → 고객에게 문자·카카오로 공유
2. 고객이 C/S 페이지에서 문의 입력
3. 해당 접수 상태 변경 확인 후 처리

### 기간 필터
최근 3개월 / 월별 / 날짜 직접 지정

### C/S 페이지 미리보기
고객이 실제 보는 화면을 미리 확인 가능합니다.

## 10. 페이백·리뷰

**경로**: 서비스접수 → 사이드 메뉴 「페이백/리뷰」  
**URL**: `/admin/inquiries/review-payback`

청소 완료 후 고객이 리뷰 작성 시 현금(페이백)을 지급하는 프로세스를 관리합니다.

### 탭 구성
| 탭 | 의미 |
|----|------|
| 전체 | 모든 신청 내역 |
| 신청접수 | 고객이 방금 신청한 건 (처리 대기) |
| 리뷰확인 | 담당자가 리뷰 확인 중인 상태 |
| 입금완료 | 페이백 금액 입금 처리 완료 |
| 반려 | 조건 미충족·허위 등으로 거절 |
| 미확인만 | 아직 확인하지 않은 신청만 표시 |

### 처리 순서
신청접수 확인 → 리뷰 링크 클릭하여 실제 리뷰 확인 → **「리뷰 확인」** 클릭 → 계좌번호 확인 후 입금 → **「입금 완료」** 클릭

## 11. 광고비 — 기간 요약 지표

**경로**: 상단 메뉴 「광고비」  
**URL**: `/admin/advertising`

### 기간 탭
기간 / 직접 / 오늘 / 이번달 / 지난달 / 올해 / 지난해 + 마케터 필터

### 9개 지표 카드

| 지표 | 의미 |
|------|------|
| 총 광고비 | 선택 기간에 입력한 광고비 총액 |
| 예약완료 건수 | 광고 유입 후 예약까지 확정된 건수 |
| 미제출 발급 | 발주서를 보냈지만 고객이 미제출한 건수 |
| 취소 건수 | 예약 확정 후 취소된 건수 |
| 삭제 건수 | 내부적으로 삭제 처리된 건수 |
| 접수 매출 합계 | 예약완료 건들의 결제금액 합계 |
| **ROAS** | 접수 매출 ÷ 총 광고비. 클수록 효율 우수 |
| 건당 비용 | 총 광고비 ÷ 예약완료 건수 (1건당 평균 광고비) |
| 일평균 광고비 | 선택 기간 일수로 나눈 하루 평균 광고비 |

> 💡 **ROAS 해석**: ROAS 10 이상이면 우수한 성과. 채널별·기간별로 비교해 효율이 낮은 채널을 점검하세요.

## 12. 광고비 — 사용자별 집계

기간 요약 카드 아래에 마케터별 성과 테이블이 나타납니다.

### 테이블 컬럼
이름 / 역할 / 광고비 / 예약완료 / 미제출 / 취소 / 삭제 / 매출 / ROAS / 건당 비용 / 일평균 광고비 / **일별 정산**

### 「일별」 버튼
각 마케터 행 우측 **「일별」** 버튼 클릭 → 날짜별 광고비 상세 팝업 (날짜·채널·금액·건수)

### 마케터 필터 연동
상단 마케터 필터에서 특정 마케터 선택 → 기간 요약 카드도 해당 마케터 데이터만 표시

## 13. 광고비 — 작업 종료·이력

### 광고비 입력 방법 (매일 업무 종료 후 필수)

**대시보드** → 텔레마케팅·광고비 블록 → **「🔴 작업 종료 및 정산」** 버튼 클릭

1. 채널별 팝업 양식이 열립니다
2. 채널(숨고·네이버·기타 등)별로 해당 날짜에 사용한 금액 입력
3. 과목(항목) 단위로 건수와 금액을 입력 가능
4. **「종료」** 클릭 → 정산 기록 저장

### 작업 종료 이력 확인
광고비 페이지 하단에서 날짜별·마케터별 입력 이력 확인

| 컬럼 | 내용 |
|------|------|
| 종료 시각 | 작업 종료 버튼을 누른 시각 |
| 담당 | 해당 마케터 이름 |
| 채널별 금액 | 채널별 금액 + 과목 건수 내역 |

> 💡 하루에 여러 번 입력 가능합니다. 이력은 모두 시각순으로 기록됩니다.

## 14. 광고비 설정

**경로**: 광고비 → 좌측 사이드 「설정」  
**URL**: `/admin/advertising/settings`

### 광고 채널 관리
- **채널 추가** — 「+ 채널 추가」로 새 채널 등록
- **채널 수정** — 채널명 변경
- **채널 비활성화** — 삭제하지 않고 사용 중지 (비활성화 채널은 작업 종료 팝업에 미표시)
- **채널 삭제** — 완전 삭제

### 정산 방식
| 방식 | 설명 |
|------|------|
| 건당 금액 | 과목별 건수 입력 시 금액 자동 계산 |
| 직접 입력 | 매번 금액 직접 입력 (유동적 채널에 적합) |

### 과목(항목) 관리
건당 금액 방식 채널에 과목을 추가합니다. 과목명 + 건당 금액 설정 후 저장.
