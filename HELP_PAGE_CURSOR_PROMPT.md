# HelpPage.tsx 작성 — Cursor용 프롬프트

아래 프롬프트를 Cursor에 복사해서 붙여넣으면 됩니다.

---

## 📋 요청 내용

청소비서 앱의 `/help` 경로에 도움말 페이지(HelpPage.tsx)를 작성해줘.

## 📊 데이터 소스

- 경로: `client/public/help/data.json`
- 구조: 배열 of `{ role, module, moduleOrder, title, path, screenshotFile, summary, markdown }`

데이터는 이미 생성되어 있고, 현재 8개 팀장 화면에 대한 설명이 들어있음.

## 🎨 UI/UX 요구사항

### 레이아웃
```
┌─────────────────────────────────────────┐
│  🏠 도움말 / Help                        │
├────────┬──────────────────────────────┤
│ Role   │                              │
│ 탭     │   Main Content Area          │
│        │                              │
│ Admin  │  Module Sections             │
│ Team   │  + Screenshots (placeholder) │
│        │  + Description               │
│ 검색창  │                              │
│        │                              │
└────────┴──────────────────────────────┘
```

### 주요 컴포넌트

1. **상단 헤더**
   - 타이틀: "도움말" 
   - 설명: "청소비서 사용 방법을 화면별로 안내합니다"

2. **좌측 사이드바** (데스크톱) / **상단 탭** (모바일)
   - 역할 선택: Admin / Team (기본: Team)
   - 검색 입력창
     - 실시간 필터링 (title, summary, markdown 기준)
     - 검색 결과 개수 표시
   - 모듈 네비게이션 (선택한 역할의 모듈만 표시)
     - 모듈 이름으로 섹션 이동

3. **메인 콘텐츠 영역**
   - 모듈별로 섹션 구분
   - 각 화면 카드:
     ```
     ┌──────────────────────────┐
     │ 제목: "오늘 일정"         │
     ├──────────────────────────┤
     │ [스크린샷 플레이스홀더]   │
     │ (800x600 회색 박스 + 텍스트) │
     │ "team_01_dashboard_main.png" │
     ├──────────────────────────┤
     │ 요약:                    │
     │ "팀장이 로그인 후..."    │
     ├──────────────────────────┤
     │ [마크다운 렌더링된 설명]  │
     │ - ## 화면 소개            │
     │ - ## 주요 기능           │
     │ - ## 사용 방법           │
     │ - ## FAQ                 │
     └──────────────────────────┘
     ```

4. **이미지 플레이스홀더**
   - 실제 이미지가 없을 때:
     - 배경: #e5e7eb (회색)
     - 중앙에 텍스트: `📷 [파일명]`
     - 크기: 100% 너비, aspect-ratio 16:9
   - 클릭 가능하게 hover 효과 추가
   - 실제 이미지가 있으면 그대로 표시

5. **마크다운 렌더러**
   - 구현 범위:
     - `## 제목` → `<h2>` 태그
     - `**굵은텍스트**` → `<strong>` 태그
     - `- 항목` → `<ul><li>` 리스트
     - `**Q: 질문**` → FAQ 스타일 (회색 배경)
     - `A: 답변` → 일반 텍스트
   - 간단한 정규식으로 충분함 (외부 라이브러리 불필요)

## 🎯 기능 요구사항

### 역할 필터
- 탭 선택: Admin / Team
- 선택한 역할의 화면만 표시
- 기본값: Team

### 검색
- 입력 필드에서 실시간 검색
- 매칭: title, summary, markdown 포함 여부
- 대소문자 구분 안 함
- 결과: "(3 results)" 형식으로 표시
- 검색 결과가 없으면: "검색 결과 없음"

### 모듈 네비게이션
- 선택한 역할의 모듈 목록 표시
- 모듈명 클릭 시 해당 섹션으로 스크롤
- 현재 보이는 모듈 하이라이트 (진한 색)
- 스크롤 시 자동 추적

### 반응형
- **데스크톱** (lg 이상): 좌측 사이드바 + 메인 콘텐츠 (비율: 1:3)
- **태블릿/모바일**: 
  - 상단 탭으로 역할 선택
  - 검색창
  - 전체 너비 콘텐츠

## 🛠 기술 스택

- **라우팅**: React Router v7 (이미 설정됨)
- **스타일**: TailwindCSS (기존 스타일 통일)
- **상태 관리**: useState (간단한 검색/필터링)
- **이미지**: `client/public/help/screenshots/` 폴더의 PNG

## 📁 파일 구조

```
client/src/pages/
  └── HelpPage.tsx          ← 생성할 파일

client/public/help/
  ├── data.json            ← 이미 있음
  └── screenshots/
      ├── team_01_dashboard_main.png
      ├── team_02_assignments_list.png
      └── ... (나중에 사용자가 추가)
```

## 🔧 라우팅 설정

App.tsx에 다음을 추가해줘:

```jsx
import { HelpPage } from './pages/HelpPage';

// Route 추가:
<Route path="/help" element={<HelpPage />} />
```

이 라우트는 **로그인 필요 없음** (공개 페이지)
