Place the Miso Partner APK here (do not commit to git).

File name example: miso-partner.apk

Package: com.miso.cleaner (Miso Partner app on Google Play)

How to get the APK:
1. Partner phone that already has the app: USB debug, then run export-apk-from-phone.bat
2. Ask Miso partner support for an official install file
3. Or use a Google Play AVD (see docs/07-android-studio-poc-setup.md section 4)

Then run install-miso-apk.bat
