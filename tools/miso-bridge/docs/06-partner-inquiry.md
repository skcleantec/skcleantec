## 보낼 질문 (초안)

**제목 예:** `[제휴 문의] 청소 업체 CRM 연동 (채팅·견적·고용 상태)`

1. **파트너(사업자)용 REST/Webhook API**가 있는지, 문서·샌드박스 제공 여부
2. 제공 범위: **채팅 목록**, **메시지 송수신**, **견적/고용/예약 상태**, **고객 연락처**(고용 후)
3. 인증 방식 (OAuth, API Key, 사업자 계정 위임)
4. Rate limit, 실시간 알림(웹훅) 지원 여부
5. **CRM·콜센터 연동** 사례 또는 공식 파트너 프로그램
6. 이용약관상 **합법적인 연동 방식** (공식 API vs 비공식 자동화)

## 우리 쪽 소개 (한 단락)

- 서비스명: **청소비서** (cbiseo.com) — 입주·이사 청소 업체용 CRM
- 이미 **숨고** 채널은 상담사 PC 로컬 브릿지로 연동 중
- 미소 채널도 **상담사 1계정 = PC 1대** 전제, 고객 데이터는 **해당 업체 테넌트**에만 저장 (멀티테넌트 SaaS)
